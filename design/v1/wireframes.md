## 공통 컴포넌트

### 토스트 알림 (3초 자동 사라짐)
```
┌─────────────────────────────────┐
│ ❤ 하트를 보냈어요                │
│ 상대방이 누구인지는 매칭 시 공개 │
└─────────────────────────────────┘
```

### 매칭 성공 모달 (2차 마감 후 자동 팝업)
```
┌───────────────────────┐
│                        │
│     🎉 💝 🎉           │
│                        │
│    매칭 성공!          │
│                        │
│       ◎                │
│     라이트 #02          │
│                        │
│  서로 하트를 보냈어요    │
│                        │
│  [쪽지 보내기]          │
│  [닫기]                 │
│                        │
└───────────────────────┘
```

### 하단 네비게이션 (참여자용, 5개 탭)
`🏠 홈` `👥 참여자` `💌 쪽지` `❤ 매칭` `⋯ 더보기(설정/요청)`

활성 탭: primary 컬러 아이콘 + 라벨 굵게
빨간 닷: 새 쪽지/매칭 알림 있을 때
