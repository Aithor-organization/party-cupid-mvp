# Wireframes v2 — Day 3 Critical Path (9 pages)

> **v1 → v2 변경점**:
> 1. 1·2차 투표 **상태 모델 UI화** (내 선택 n명 / 단계 배지 / 유지·신규 표시)
> 2. 하트 **오동작 방지** (탭존 분리, 취소 바텀시트, 마감 직전 경고)
> 3. 안전 맥락 강화 (상세·쪽지 신고 1차 액션, 차단 vs 신고 구분)
> 4. 엣지 상태 (검색 empty, 로딩 skeleton, 전송 실패 재시도)
> 5. 주최자 방상세 — 현재 단계에 따라 primary 액션 1개만, danger 영역 분리
> 6. 접근성 반영 (aria-label, 색+텍스트 병행, 포커스, reduced-motion)

### 상태별 primary 액션 1개만 노출

```
┌─────────────────────────────────────────┐  1280×800
│ ← 대시보드 │ 2026 봄 파티  🟢 LIVE        │
├─────────────────────────────────────────┤
│                                          │
│ ┌─── QR 코드 ────┐  ┌─── 라이브 현황 ───┐│
│ │                │  │ 참여자  23 / 50    ││
│ │   ██▀██▀▀█    │  │ ▓▓▓▓▓░░░░░ 46%     ││
│ │   █▄ ██ ▄█    │  │                    ││
│ │   ▀▀█▄▄█▀▀    │  │ 단계  💜 2차 진행 중 ││
│ │                │  │ 마감  00:42:11      ││
│ │ party.app/r/   │  │                    ││
│ │    ABC123      │  │ 하트 누적  147      ││
│ │                │  │ 예상 매칭  ~12      ││
│ │ [PNG 다운]     │  │                    ││
│ │ [URL 복사]     │  │                    ││
│ └────────────────┘  └────────────────────┘│
│                                            │
│ ─── 다음 액션 ───                          │
│ ┌────────────────────────────────────────┐│  ← primary 1개만
│ │ 💜 2차 투표 진행 중                      ││
│ │ 마감까지 자동 진행됩니다.                ││
│ │ [조기 마감하기]   (secondary)            ││
│ └────────────────────────────────────────┘│
│                                            │
│ ─── ⚠ 위험 액션 ───                        │  ← 분리된 danger 영역
│   [파티 강제 종료] (danger)                │
│                                            │
│ ─── 탭 ───                                 │
│ [참여자 23] [신고 2 🔴] [요청 5] [결과]    │
│                                            │
└─────────────────────────────────────────┘
```

**상태별 primary 액션 규칙**:

| 파티 상태 | primary 액션 | 설명 |
|----------|-------------|------|
| 시작 전 | `[지금 시작]` | 예정 시간 전이라도 즉시 시작 가능 |
| 시작됨, 투표 전 | `[🌸 1차 투표 시작]` | 지정 시간에 자동 / 수동 override |
| 1차 진행 중 | `[조기 마감]` secondary only | primary 없음, 자동 진행 |
| 1차~2차 사이 | `[💜 2차 투표 시작]` | 지정 시간에 자동 / 수동 override |
| 2차 진행 중 | `[조기 마감]` secondary | 같음 |
| 투표 완료 | `[파티 종료]` | 결과 공개 후 마무리 |
| 종료됨 | (액션 없음) | 결과 뷰만 |

**위험 액션 (파티 강제 종료 / 참여자 전원 강퇴)** 은 항상 **별도 danger 영역**에 분리, 빨간 텍스트 + 확인 모달 2단계.
