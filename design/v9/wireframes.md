## 충돌 diff 모달 (v5 신규 — 하트 저장 충돌 시)

자동 새로고침 대신 사용자 선택:

```
┌──────────────────────────────┐
│  ⚠ 다른 기기에서 변경됐어요     │
│                                │
│  02 라이트에 대한 변경:          │
│                                │
│  ━━ 내가 방금 한 변경 ━━        │
│  + ❤ 선택                       │
│  (22:06:15)                    │
│                                │
│  ━━ 다른 기기에서의 변경 ━━     │
│  - ❤ 취소                       │
│  (22:06:42 · 27초 전)           │
│                                │
│  어떻게 할까요?                  │
│                                │
│  ┌──────────────────────────┐ │
│  │ 내 변경 유지               │ │
│  │ (선택 상태로 저장)          │ │
│  └──────────────────────────┘ │
│  ┌──────────────────────────┐ │
│  │ 다른 기기 유지              │ │
│  │ (취소 상태로 저장)          │ │
│  └──────────────────────────┘ │
│  ┌──────────────────────────┐ │
│  │ 제자리 유지 (나중에 결정)   │ │
│  │ (현재 화면 그대로, 이 하트  │ │
│  │  만 ⏳ pending 상태로 표시) │ │
│  └──────────────────────────┘ │
│                                │
│  ℹ ESC 키 누르면 "제자리 유지"  │  ← v7: ESC 허용 (접근성)
│    선택과 동일하게 처리됩니다    │
│                                │
└──────────────────────────────┘
```

접근성: `role="alertdialog"`, 첫 버튼 autofocus, **ESC 허용** — "제자리 유지"와 동일 처리 (접근성 탈출 경로 보장). `aria-describedby`로 ESC 동작 명시.

## 변경 요약 (v3/v4 → v5)

| 영역 | v3 | v4 | v5 |
|------|-----|-----|-----|

| **포커스 규칙** | 기초 | Patch 5/6 충돌 | **단일 표로 확정** |
| **DS 통합** | 베이스 | patch 분리 | **ROUND_BANNER/HEART_SAVE_STATE 정식 등록** |
| **온보딩** | 없음 | 없음 | **4-step + 재보기** |
| **신고 후** | 성공 토스트만 | (v4 동일) | **정책 설명 + 즉시 차단 경로** |
