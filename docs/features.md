## 강퇴 시 모수 갱신 시퀀스 (v4)

```
[주최자] 참여자 X 강퇴 클릭
   ↓
1. Participant.status = 'kicked'
2. Participant.kicked_at = now
3. SSE 브로드캐스트: { type: 'participant-kicked', target: X.id }
   ↓
모든 참여자 클라이언트:
   - X 카드 즉시 제거
   - 만약 X에게 ❤ 보냈었다면 → "X님이 강퇴됐어요. 보낸 ❤ 무효 처리" 토스트
   ↓
서버 모수 재계산:
   - 현재 단계의 전원완료 카운터: 23/23 → 22/22 (X 제외)
   - 만약 X 제외로 즉시 100% 도달 → 단계 자동 종료
   ↓
DB 업데이트:
   - Vote 테이블에서 X.id 관련 레코드는 보존 (기록 영구)
   - is_locked 유지 (영구 잠금 정책)
```

## 트리거 충돌 5초 카운트 모달 (v4)

```typescript
// 주최자 [다음 단계로] 클릭 시
function handleManualAdvance(stageId: string) {
  // 5초 카운트 모달 열기
  const modal = openCountdownModal({
    title: '다음 단계로 이동',
    body: `5초 후 강제 마감됩니다. 현재 18/23 명 투표 완료`,
    countdown: 5,
    onCountdownComplete: () => closeStage(stageId),
    onAccept: () => closeStage(stageId),  // [지금 마감]
    onCancel: () => null,                  // [취소]
  });

  // 중간에 다른 트리거 발동 (타이머/전원완료) → 모달 닫고 자동 종료
  watchOtherTriggers(stageId, () => {
    modal.close();
    closeStage(stageId);
  });
}
```
