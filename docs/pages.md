### A-4 닉네임 등록 + Anonymous Auth (`/r/:code/nickname`) ⭐ v6 강화

```
┌───────────────────────┐  모바일: 375×812
│  ← 뒤로                │
│                        │
│  🎉 지은의 생일파티      │
│  방에 들어가기          │
│                        │
│  닉네임을 입력해 주세요  │
│  (2~12자, 한글/영문/숫자)│
│  ⬚ _________________   │
│                        │
│  ⓘ 같은 닉네임이 있으면  │
│  자동으로 (2)가 붙어요  │
│                        │
│  ─────────────────    │
│                        │
│  💡 한 기기에서 계속     │
│  사용해 주세요. 다른     │
│  기기에서는 다시 입장   │
│  해야 해요              │
│                        │
│  [입장하기]            │
└───────────────────────┘

[입장하기] 클릭 시 동작:
1. supabase.auth.signInAnonymously() 호출 → anonymous JWT 발급
2. participants 테이블에 INSERT:
   - room_id = :code → id 변환
   - anon_user_id = jwt.sub
   - nickname = 입력값
   - entry_number = 자동 채번
3. localStorage에 refresh_token 자동 저장 (Supabase SDK 기본)
4. → /r/:code/home 이동

세션 복구 (다음 방문 시):
- A-1 (/r/:code) 진입 시 supabase.auth.getSession() 확인
- 세션 있고 같은 room_id의 participant row 존재 → 바로 /home
- 세션 없거나 다른 방 → /consent부터 다시
```
